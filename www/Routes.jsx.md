# [Routes.jsx](Routes.jsx)

Routes constructed here use .zland config add this to your .zland json config to create a route for your project

Example:
```json
 "routes": [
    {
      "path": "intro",
      "handler": "components/IntroductionWindow"
    }
 ]
```
